# HibernateCrudRepository
Web service (REST) application that allows to made CRUD operations in DB.

## 1. Technologies
1. SpringWebMVC 6.0.9 + Tomcat 10.1.9
2. Java 17
3. PostgreSQL
4. Hibernate
5. Lombok

## 2. Database
1. Employee
   * first_name
   * last_name
   * position
2. Position 
   * name
3. Project 
   * name
4. Employee_project
   * employee_id 
   * project_id
